# UX Study Summary Example

## Scenario
A professional user wants a reusable example that shows how to ask for useful agent output without exposing private data.

## Example Request
Request: Summarize a task-based usability review of a service page with task success issues, confusing labels, accessibility concerns, and prioritized fixes.

## Good Output Should Include
- A concise summary of the task and source basis.
- Clear assumptions and limitations.
- Practical recommendations or next steps.
- Privacy-safe language and no sensitive details.

## Review Notes
This example is intentionally generic. Replace the scenario with sanitized project context before using it in real work.
