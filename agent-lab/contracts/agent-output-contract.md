# Agent Output Contract

## Purpose
Standard fields: summary, what was analyzed, evidence or source basis, assumptions, findings, risks, recommendations, limitations, next steps, approval needed if applicable, and handoff target if applicable.

## Required Fields
- Goal or artifact name
- Source basis
- Constraints
- Risk level
- Approval status

## Optional Fields
- Deadline or priority
- Owner
- Related files
- Follow-up review date

## Validation Checklist
- Required fields are complete
- Assumptions are labeled
- Sensitive data is excluded or approved
- Approval status is clear
- Output can be handed to another agent or reviewer

## Example
Use this contract to capture the minimum safe context before starting or transferring work.
