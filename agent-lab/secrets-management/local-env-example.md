# Local Environment Example Guide

This is not a real `.env` file. Do not paste real secrets into this file. Copy this pattern into `.env.local` or `.env` locally, but never commit real secret values.

```text
API_BASE_URL=https://example.com/api
PUBLIC_SITE_NAME=Example Site
NETLIFY_SITE_ID=NETLIFY_SITE_ID_PLACEHOLDER
DATABASE_URL=DATABASE_URL_PLACEHOLDER
SERVICE_API_KEY=YOUR_API_KEY_HERE
```
