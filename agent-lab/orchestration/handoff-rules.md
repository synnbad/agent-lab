# Handoff Rules

## Required Handoff Information
- Source agent and target agent
- Reason for handoff
- Goal and expected output
- Context summary and source files
- Decisions already made
- Constraints and assumptions
- Risks and approval status
- Open questions

## Avoiding Context Loss
Compress context into a short source-of-truth summary, preserve links or file paths, and label unresolved questions. Do not hand off raw sensitive data unless approved and necessary.
