# Routing Rules

## Task Routing
- Data questions route to Data Analysis Agent.
- Source comparison routes to Research Agent.
- Reports and polished communication route to Writing Agent.
- Accessibility review routes to Accessibility Agent.
- Automation planning routes to Automation Agent.
- Web app and frontend planning routes to Web Development Agent.
- Interface structure routes to UI Agent.
- API and backend planning routes to Backend Agent.
- Architecture decisions route to System Design Agent.
- Client website delivery routes to Client-to-MVP Website Agent.
- Library assessment work routes to Library Assessment Specialist Agent.
- Dashboard work routes to Dashboard Reporting Agent.
- Survey work routes to Survey Assessment Agent.
- UX assessment routes to UX Assessment Agent.
- Data privacy and governance routes to Data Governance Privacy Agent.
- Leadership communication routes to Stakeholder Insight Agent.

## Similar Agent Choices
Choose the narrowest agent that can complete the work. Use System Design Agent for architecture tradeoffs, Backend Agent for API details, and Web Development Agent for full-stack delivery planning.

## Grill-Me Sinbad
Invoke Grill-Me Sinbad before execution when the plan is high-risk, expensive, vague, client-facing, stakeholder-facing, deployment-related, or dependent on several uncertain assumptions.

## Escalation
Escalate to Data Governance Privacy Agent when sensitive data, access controls, small groups, or public reporting are involved.
