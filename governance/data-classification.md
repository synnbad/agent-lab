# Data Classification

| Class | Description | Examples | Allowed Agent Use | Required Approval | Storage Rules |
|---|---|---|---|---|---|
| Public | Safe to share openly | Published docs, public website copy | Allowed with review | None for low-risk use | May be stored in repo |
| Internal | Routine non-public work material | Process notes, internal templates | Use with caution | Approval for external sharing | Store only in approved systems |
| Confidential | Sensitive business or institutional context | Private planning notes, client drafts | Use only when necessary | Explicit approval | Do not store in public repo |
| Restricted | Highly controlled data | Access-controlled reports, private records | Avoid unless required | Explicit owner approval | Approved secure systems only |
| Sensitive regulated data | Data with legal, privacy, or safety risk | Student, staff, patron, HR, medical, financial, immigration, or identifying data | Prefer not to use; use aggregate or redacted forms | Formal approval | Never store in this repo |
