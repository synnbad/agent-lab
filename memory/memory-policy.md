# Memory Policy

Agents may retain reusable process knowledge, safe preferences, and public project patterns. Project context and run context are temporary. Never save private workplace data, raw client content, private student, staff, or patron data, HR records, medical information, immigration data, credentials, or secrets. Durable memory is reusable and non-sensitive; project context belongs to a project; run context belongs to the current task; sensitive data must not be retained.

## Checklist
- Source basis identified
- Sensitive data excluded
- Assumptions labeled
- Next step clear
